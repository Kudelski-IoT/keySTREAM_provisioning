Start the FOTA download and installation process.

Sets the FOTA state to IN_PROGRESS, saves the FOTA name and metadata, and prepares all target components for update. This function should be called to trigger a FOTA update. The actual download and install may be handled by a platform-specific thread or task.

Parameters
[in]xpFotaNamePointer to the FOTA name (must not be NULL).
[in]xFotaNameLenLength of the FOTA name.
[in]xpFotaMetadataPointer to the FOTA metadata (must not be NULL).
[in]xFotaMetadataLenLength of the FOTA metadata.
[in]xpTargetComponentsArray of target components (must not be NULL).
[out]xComponentsArray to receive the current installed components.
Returns
E_K_FOTA_SUCCESS, E_K_FOTA_IN_PROGRESS, or E_K_FOTA_ERROR.

Start the FOTA download and installation process.

Gets the status of the FOTA update.

Overall Fota update status will be provided to caller. Upon failed or successful installation, structure components will be updated.

Parameters
[in]xFotaNamePointer to the FOTA name. Should not be NULL.
[in]xFotaNameLenLength of the FOTA name.
[out]xFotaErrorPointer to the FOTA error structure. Should not be NULL.
[out]componentsArray of currently installed components.
Returns
  • E_K_FOTA_SUCCESS in case of success.
  • E_K_FOTA_IN_PROGRESS in case of FOTA in progress.
  • E_K_FOTA_ERROR for other errors.

Gets the status of the FOTA update.
