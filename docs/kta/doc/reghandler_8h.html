keySTREAM Trusted Agent - Registration module interface.

Build registration request.

Parameters
[in]xpRecvdProtoMessageShould not be NULL. Message received from application.
[out]xpMessageToSendShould not be NULL. Buffer should be provided by caller of at-least C_K__ICPP_MSG_MAX_SIZE size. Buffer will be filled with icpp Message to send to keySTREAM.
[in,out]xpMessageToSendSizeShould not be NULL. BufferSize will be filled with icpp Message size to be send to keySTREAM. If it is 0 then it means no data to send to server.
Returns
  • E_K_STATUS_OK in case of success.
  • E_K_STATUS_PARAMETER for wrong input parameters.
  • E_K_STATUS_ERROR for other errors.
